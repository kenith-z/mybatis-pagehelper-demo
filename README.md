# SpringBoot2.5.3整合Mybatis3.5.7基于PageHelper的分页

这是一个基于SpringBoot2.5.3整合MyBatis3.5.7使用PageHelper实现分页的极简教程，笔者使用到的技术及版本如下：

1. SpringBoot 2.5.3
2. Mybatis 3.5.7
3. PageHelper 5.2.1

写博客的起因是某位程序猿小姐姐在使用低代码平台时，跑通了数据查询功能后尝试对数据进行分页发现不起作用，所以笔者花了1个小时快速实现了整合demo供小姐姐研究学习。
